# Breadth-First Search
## 簡介
Breadth-First Search(BFS，廣度優先搜尋)，是廣義的Level-Order Traversal，同樣的原理使用在 Graph，就叫做 BFS 。他是已同層次(level)的節點優先，由低層次到高層次。
